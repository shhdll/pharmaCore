# PharmaCore

PharmaCore is a React + Vite pharmacy dashboard that simulates inventory, prescriptions, and patient management.

## Features
- Medication inventory tracking (stock, expiry, status)
- Prescription management (active, dispensed, pending)
- Patient profiles with basic medical data
- Admin, Pharmacist, and Patient views
- Simple analytics dashboard

## Tech Stack
React, Vite, Tailwind CSS, Framer Motion, Lucide Icons

## Setup
```bash
npm install
npm run dev
